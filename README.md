# Bootstrap 5 Crash Course

This project is a hands-on crash course designed to learn and practice the fundamentals of the Bootstrap 5 framework. It covers essential components, the grid system, and utility classes to build modern, responsive web pages.

## Project Overview

The goal of this project is to build a simple, clean landing page for a fictional "Frontend Bootcamp." This allows for the practical application of various Bootstrap components in a real-world context.

## Features Implemented

As the project progresses, it will include the following Bootstrap features:

-   **Responsive Navbar:** A fully responsive navigation bar that collapses on smaller screens.
-   **Grid System:** Use of Bootstrap's powerful mobile-first flexbox grid to create complex layouts with columns and rows.
-   **Showcase/Hero Section:** A prominent introductory section to grab the user's attention.
-   **Cards:** Used for displaying content in a structured and visually appealing way.
-   **Buttons & Forms:** Customizing and styling interactive elements.
-   **Utility Classes:** Leveraging Bootstrap's utility API for spacing, colors, borders, and more to speed up development.

## Technologies Used

-   **HTML5**
-   **CSS3**
-   **Bootstrap 5**

## How to Use

To view or run this project locally, simply follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/loganghy/bootstrap-crash-course-.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd bootstrap-crash-course-
    ```
3.  **Open the `index.html` file in your web browser.**

There are no build steps or dependencies required.
